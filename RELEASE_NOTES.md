# 发布说明 - v1.0.0

**发布日期**: 2025年12月26日

## 📦 版本概述

这是员工健康证管理系统的第一个正式版本，包含完整的功能模块和文档。

---

## ✨ 主要功能

### 1. 用户认证与授权
- ✅ 管理员/员工双角色登录
- ✅ JWT Token认证机制
- ✅ 密码BCrypt加密存储

### 2. 健康证管理
- ✅ 健康证上传（支持JPG/PNG/BMP格式）
- ✅ OCR自动识别（PaddleOCR集成）
- ✅ 健康证审核（通过/拒绝）
- ✅ 健康证查询和历史记录
- ✅ 已拒绝健康证可重新上传

### 3. 员工管理
- ✅ 员工信息查询（分页）
- ✅ 员工信息编辑（手机号、钉钉ID等）
- ✅ 员工数据同步（从HR系统）

### 4. 部门管理
- ✅ 部门树状结构显示
- ✅ 显示主管姓名和部门人数
- ✅ 部门启用/禁用功能
- ✅ 支持展开/折叠

### 5. 数据看板
- ✅ 统计卡片（6个关键指标）
- ✅ 数字颜色显示（蓝色/绿色/黄色/红色/紫色）
- ✅ 下钻功能（点击统计卡片查看详细员工列表）
- ✅ 状态分布饼图（颜色与统计卡片一致）
- ✅ 部门统计柱状图

### 6. 系统配置
- ✅ 邮件配置（SMTP设置、测试发送）
- ✅ 钉钉配置（Corp ID、App Secret）
- ✅ 提醒规则配置（到期提醒天数）
- ✅ 手动触发同步（员工/部门）

### 7. 提醒功能
- ✅ 健康证到期提醒（邮件/钉钉）
- ✅ 定时任务（每天上午9点）

### 8. 导出功能
- ✅ Excel格式导出
- ✅ PDF格式导出
- ✅ 数据筛选导出

### 9. 操作日志
- ✅ 用户操作记录
- ✅ 日志查询和筛选

---

## 🛠️ 技术特性

- **后端**: Spring Boot 2.7.18 + MyBatis-Plus 3.5.3
- **数据库**: MySQL 8.0（双数据源：本地+远程HR系统）
- **前端**: 纯HTML/CSS/JavaScript（ES6+）
- **部署**: Docker + Docker Compose + Nginx
- **认证**: JWT Token
- **OCR**: PaddleOCR集成
- **可视化**: Chart.js

## 🚀 快速开始

### 前置要求

- Docker 20.10+
- Docker Compose 1.29+
- 4GB+ 内存
- 10GB+ 磁盘空间

### 部署步骤

```bash
# 1. 克隆项目
git clone https://github.com/AprilZB/AP-health_cert.git
cd health-cert-system

# 2. 启动服务
docker-compose up -d

# 3. 等待服务启动（约10-20分钟）
docker-compose logs -f

# 4. 访问系统
# http://localhost
```

### 默认账号

- **用户名**: `admin`
- **密码**: `admin123`

> ⚠️ **安全提示**: 首次登录后请立即修改默认密码！

## 🐛 已知问题

1. **OCR识别大图片可能超时**
   - 影响: 中等
   - 解决方案: 压缩图片或增加超时时间

2. **图片预览在某些浏览器不支持**
   - 影响: 低
   - 解决方案: 使用Chrome/Firefox/Edge浏览器

3. **分页在某些情况下不准确**
   - 影响: 低
   - 解决方案: 刷新页面或使用筛选条件
